🎉 Let's Learn HTML & CSS Tables!

Tables help us organize data. With HTML, we create the structure. With CSS, we style it to look awesome!

📐 Table Sizes

Set table width using the width attribute or CSS property:

<table style="width: 50%;">...</table>

📌 Table Headers

Use <th> for header cells:

<tr>
  <th>Name</th>
  <th>Age</th>
</tr>

📏 Padding & Spacing

Control spacing with CSS:

td {
  padding: 15px;
}

🧩 Colspan & Rowspan

Merge cells across columns or rows:

<td colspan="2">Merged Cell</td>
<td rowspan="2">Merged Row Cell</td>

🎨 Table Styling

Apply CSS to beautify your tables:

table {
  border: 2px dashed #ff69b4;
  background-color: #fff0f5;
}

📊 Table Colgroup

Style groups of columns:

<colgroup>
  <col span="2" style="background-color: lavender;">
  <col style="background-color: lightblue;">
</colgroup>
